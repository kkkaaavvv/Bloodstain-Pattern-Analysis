# 🩸 Automated Bloodstain Pattern Analysis and Crime Scene Reconstruction

A computer vision-based forensic analysis system that automates bloodstain detection, geometric analysis, and trajectory reconstruction to estimate the area of origin.

---

## 🚀 Overview

This project implements an end-to-end pipeline for Bloodstain Pattern Analysis (BPA) using image processing and geometric modeling. The system reduces manual effort by providing automated, consistent, and data-driven analysis.

---

## 👩‍💻 My Contribution

This was a team project. My primary contributions include:

* Developed the frontend interface using React for image upload and result visualization
* Integrated frontend with backend APIs for seamless data flow
* Worked on image preprocessing and segmentation using OpenCV
* Contributed to contour detection and feature extraction pipeline
* Assisted in designing the overall system workflow and user interaction

---

## 🛠️ Tech Stack

* Python, OpenCV, NumPy
* Flask (Backend)
* React (Frontend)
* Git, VS Code

---

## ⚙️ System Pipeline

* Image preprocessing (grayscale, Gaussian blur)
* Segmentation (Otsu thresholding, morphology)
* Contour detection and filtering
* Ellipse fitting for geometric feature extraction
* Trajectory back-projection
* Area of origin estimation

---

## 📊 Results

* Precision: 74%
* Recall: 68%
* mAP: 71.9%
* F1 Score: 71%
